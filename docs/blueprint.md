# **App Name**: MetalChat

## Core Features:

- User Authentication: User authentication and management using Clerk, with API key integration.
- Real-Time Messaging: Real-time messaging functionality to send text messages between users.
- Media Sharing: Ability to share images, videos, and sound recordings in chats.
- Chat Summarization: AI tool that can summarize chat conversations to provide a quick overview.
- threads layout: A threads like feature on user personal details

## Style Guidelines:

- The app should primarily use dark backgrounds, based on a Dark color scheme.
- Primary color: Deep Charcoal (#333333) to create a sophisticated and modern look, suggesting the "metal" theme and evoking a sense of technological advancement (HSL: 0, 0%, 20%).
- Background color: Near Black (#1A1A1A) with minimal saturation complements the Charcoal without causing excessive contrast (HSL: 0, 0%, 10%).
- Accent color: Electric Purple (#BE70FF) provides contrast and draws attention to interactive elements. (HSL: 274, 100%, 74%).
- Use clean sans-serif fonts for a modern and readable interface.
- Incorporate minimalist, outlined icons with a metallic finish to align with the theme.
- Implement subtle transparency effects to add depth and a modern touch to the UI.
- Introduce smooth animations for transitions and loading states to enhance the user experience.
- Thread like elements should also use smooth animations when being added.
